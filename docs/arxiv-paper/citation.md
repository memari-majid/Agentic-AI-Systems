# How to Cite

## Citing the Review Paper

If you use this comprehensive review paper in your research or work, please cite it as follows:

### BibTeX Format

```bibtex
@article{memari2025agentic,
  title={Agentic AI Systems: A Comprehensive Framework for Building Autonomous Intelligent Agents},
  author={Memari, Majid},
  journal={arXiv preprint arXiv:XXXX.XXXXX},
  year={2025},
  note={Available at: https://github.com/memari-majid/Agentic-AI-Systems/arxiv-paper}
}
```

### Plain Text Format

```
Memari, M. (2025). Agentic AI Systems: A Comprehensive Framework for Building 
Autonomous Intelligent Agents. arXiv preprint arXiv:XXXX.XXXXX.
```

### APA Style

```
Memari, M. (2025). Agentic AI Systems: A Comprehensive Framework for Building 
Autonomous Intelligent Agents. arXiv. https://github.com/memari-majid/Agentic-AI-Systems/arxiv-paper
```

### MLA Style

```
Memari, Majid. "Agentic AI Systems: A Comprehensive Framework for Building 
Autonomous Intelligent Agents." arXiv, 2025.
```

## Citing the Knowledge Base Repository

If you use the complete knowledge base (62 chapters, 13 labs) in your work:

### BibTeX Format

```bibtex
@misc{memari2025agenticai,
  author = {Memari, Majid},
  title = {Agentic AI Systems: A Comprehensive Knowledge Base},
  year = {2025},
  publisher = {GitHub},
  url = {https://github.com/memari-majid/Agentic-AI-Systems},
  note = {Documentation: https://memari-majid.github.io/Agentic-AI-Systems/}
}
```

### Plain Text Format

```
Memari, M. (2025). Agentic AI Systems: A Comprehensive Knowledge Base. 
GitHub. https://github.com/memari-majid/Agentic-AI-Systems
```

## Citing Specific Chapters or Labs

### For Specific Chapters

```bibtex
@misc{memari2025foundations,
  author = {Memari, Majid},
  title = {Foundations of Agentic AI Systems},
  year = {2025},
  howpublished = {Online},
  url = {https://memari-majid.github.io/Agentic-AI-Systems/01-foundations/},
  note = {Chapter from: Agentic AI Systems Knowledge Base}
}
```

### For Specific Labs

```bibtex
@misc{memari2025labs,
  author = {Memari, Majid},
  title = {Agentic AI Systems: Hands-On Laboratory Exercises},
  year = {2025},
  howpublished = {Online},
  url = {https://memari-majid.github.io/Agentic-AI-Systems/06-labs/},
  note = {13 Python implementations with tutorials}
}
```

## What to Cite

### Use the Review Paper When:
- Citing the comprehensive framework and theoretical foundations
- Referencing the comparative analysis of approaches
- Discussing the state of the field
- Academic publications and research papers

### Use the Repository When:
- Using the educational content or tutorials
- Referencing specific code implementations
- Adapting labs or examples for teaching
- Building on the practical guides
